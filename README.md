# PetAdoptionSystem
A Java console-based Pet Adoption Manangement System developed for a school project.

## Features
## User Management
- Admin and Staff login system
- User role validation
- Login and logout history tracking

## Pet Management
- Add new pets
- View all pets
- Search pets by ID
- Update pet information
- Remove pets from the system

## Adoption System
- Adopt pet
- Browse pets by type
- View available pets
- Store adopter informatin
- Save adoption history

## Reports and Records
- View recent adoptions
- View adopter list
- View user access history
- View user changes history
- Count total adopted and available pets

## Data Storage
- File handling using .txt files
- Relative file paths for portability
- Persistent data saving

# Technologies and Concepts Used
- Java
- NetBeansIDE
- File Handling
- Arrays
- Enums

# How to Run
1. Open the Project in Netbeans
2. Make sure the data folder exists in the project root
3. Ensure all .txt files are inside the data folder
4. Run PetAdoptionSystem.java
